mobilAP Documentation - Site Administration

There are many important site options you can configure in mobilAP. Some of these settings include:

  • General Settings - Such as title, password and schedule options
  • Menu List - For configuring the main menu labeling and visibility
  • Evaluation Questions - Configuring the questions used for session evaluations
  • Sessions - For administering and removing sessions
  • Content - For updating and uploading content for certain pages and images

Note that all the actions below assume you are logged in as an administrative user

General Settings

There are several settings that affect the behavior of mobilAP. To edit this settings:

  1. Click on the Admin menu
  2. Click on the Settings submenu
  3. Update settings as needed
  4. Click Save to commit your changes

The following settings can be updated:

Site Title
Enter the title of your site. It is used in the header of the site. Because of the limited space on the mobile site, you may wish to make this short or abbreviated.
Require Login to view content
You can make all the content on the site private and require users to login in order to view the site
Use Passwords
You can require attendees to enter a password when logging in. You assign passwords by editing the user in the attendee administration. By default, a user's password is their email address.
Require Password for admins
You can require admins to enter a password when logging in. This will allow attendees to login without a password, but prevent unauthorized administration without a password.
Require Password for presenters
You can require presenters to enter a password when logging in. This will allow attendees to login without a password, but prevent unauthorized administration without a password.
Allow self-created users
You can allow anyone to create an account in the system by supplying their email address. If off, you must administer all users before they can login. This is very useful for events where the attendees are not known in advance.
Use simple schedule
For simpler events where there is only 1 session, you can turn off the schedule. This will create a single session with a single set of links, questions and discussion.
Time Zone
Select the time zone to use for this event.

Menu List

You can customize the menu of mobilAP to meet the needs of your event. This includes changing the labels of menu items as well as changing the visibility of certain menu items. You can customize the following areas:

  • Welcome
  • Schedule
  • Directory
  • Announcements

Updating the menu

  1. Click on the Admin menu
  2. Click on the Home List submenu
  3. Next to each item is a text box, update the label as necessary
  4. To turn off a menu item, click the switch so it reads Off
  5. Click Save to commit your changes

At this time you cannot change the order of the menu items.

Evaluation Questions

If you wish, you can create a series of evaluation questions for each session. This allows users to rate their experience with each session and give their feedback. Questions can be multiple choice or free response. Although all sessions will use the same set of questions, you can turn off evaluations for certain sessions if you wish.

Adding an Evaluation Question

  1. Click on the admin menu
  2. Click on Evaluation Questions in the tab bar
  3. Click Add Question
  4. Enter the question text
  5. Choose the question type
    • Multiple Choice questions allow the user to pick 1 of several possible responses
    • Free Response questions will be presented with a text box where users' can type in their response (suitable for comments)
  6. For multiple-choice questions, type in a response and click Add for each possible response.
  7. Click Save to add the question

Removing an Evaluation Question

  1. Click on the admin menu
  2. Click on Evaluation Questions in the tab bar
  3. Choose an existing question from the list
  4. Click Delete to remove the question

Note: it is not recommended to add or remove evaluation questions after the event has begun and evaluations have already been submitted. This can lead to misleading results

Sessions

Each schedule item must contain a session, but a session can occur at several different points in the schedule. It is even possible to completely remove a session from the schedule.

Removing a session

Note: removing a session will completely delete all of its data including links, questions, evaluations and discussion. It will also remove it from any portion of the schedule it may be assigned to. This cannot be undone so take care when using this feature

  1. Click on the admin menu
  2. Click on Sessions in the tab bar
  3. Click the Edit button
  4. Click Delete next to the session you with to remove

Content

There is some content you can update using the administration interface. Since this involves editing files in place it requires that the web server can write to the mobilAP/data folder. If this is not possible, then you can manually update the files as needed.

Updating the mobile home screen icon

The iPhone OS supports bookmarking sites by adding them to the home screen. You can customize the icon the appears on the home screen by uploading a new image. This image must be:

  • A PNG image file
  • 57 x 57 pixels

To upload a new image:

  1. Click on the admin menu
  2. Click on Content in the tab bar
  3. In the mobile home screen icon area, browse to find the file you wish to upload
  4. Click the Upload button

You should receive appropriate feedback on success or failure
