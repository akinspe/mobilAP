Editing the Schedule

Building your schedule is one of the most critical parts of managing your event or class. You build the schedule by adding a series of meetings or sessions, each occurring at a particular time. Each item is assigned to a session, which contains the title and holds all of the user interactive data such as links, questions and discussion. To accommodate situations where you want to maintain the session data (links, discussion, etc) at different times, you can assign a session to multiple schedule items.

As a reminder, for each of these tasks you will need to be sure you are logged in as an administrative user.

Creating a new schedule item

  1. Click on the schedule menu
  2. Click the + button
  3. Choose the date, start time and end time as necessary.
  4. Choose a session or add a new session
  5. Optionally add detail information. This appears with the session title and can give additional information about the meeting
  6. Optionally add a room or location
  7. Click save to add the item to the schedule

Editing an existing schedule item

  1. Click on the schedule menu
  2. Click the Edit button. This will turn the list into edit mode
  3. Click the Edit button next to the item you wish to edit.
  4. Choose the date, start time and end time as necessary.
  5. Choose a session or add a new session
  6. Optionally update detail information. This appears with the session title and can give additional information about the meeting
  7. Optionally update room or location
  8. Click Save to update the data, Cancel to return without saving or Delete to remove the item from the schedule.

Adding a new session

You can get to the add session section by either:

  • Clicking add new session from the add schedule item section.
  • Clicking the + button from the session administration section.

From this section you can create a new session, which can then be added to the schedule.

  1. Enter a title (required). This will be the title shown in the schedule
  2. Optionally, enter a description. This can be of any length. Currently it does not support any formatting or links. Use a blank line to separate paragraphs
  3. Select the options for this session
    • Show Links - This turns on or off the link sharing feature for this session
    • Allow non-presenters to post links - If this is unchecked, only presenters and administrators will be able to post links
    • Enable Discussion - This turns on or off the discussion feature for this session
    • Enable Evaluation - This turns on or off the evaluation feature for this session. Note: if no evaluation questions have been defined for this site, the evaluation will not appear regardless of this setting.
  4. Click save to add this session. It can now be added to the schedule

To edit a session, simple click on it in the schedule.
