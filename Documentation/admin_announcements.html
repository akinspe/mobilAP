mobilAP Documentation - Announcement Administration

Announcements provide a way for site administrators to communicate important messages to users. It is useful to post things that are not part of any particular session

As announcements can only be posted by site administrators, the following steps assume you are logged in using an administrative account.

Creating an announcement

  1. Click on the announcements menu
  2. Click Add Announcement
  3. Enter the title
  4. Enter the text. No formatting is permitted. Paragraphs should be separated by a blank line
  5. Click save to create the announcement

Editing an announcement

  1. Click on the announcements menu
  2. Click on an announcement
  3. Click the Edit button
  4. Enter the title
  5. Enter the text. No formatting is permitted. Paragraphs should be separated by a blank line
  6. Click save to create the announcement

Removing an announcement

  1. Click on the announcements menu
  2. Click on an announcement
  3. Click the Delete button

Announcements are shown in chronological order
