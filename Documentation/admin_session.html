mobilAP Documentation - Session Administration

Managing a session

In order to edit session data such as questions, links or presenters, simply view the session by using the schedule. All the examples shown below assume you have logged in as a site administrator or session presenter.

You can manage:

  • Session Information and Options
  • Session Presenters
  • Links
  • Evaluations
  • Questions
  • Discussion

Session Information and Options

Updating session information and options

  • Choose the session in the schedule
  • Click the Admin Tab
  • Update the title as needed, this will affect the session's title any place it is located in schedule.
  • Update the description as needed. The description is visible in the Info tab of the session
  • Update options as needed:
    • Show Links - This turns on or off the link sharing feature for this session
    • Allow non-presenters to post links - If this is unchecked, only presenters and administrators will be able to post links
    • Enable Discussion - This turns on or off the discussion feature for this session
    • Enable Evaluation - This turns on or off the evaluation feature for this session. Note: if no evaluation questions have been defined for this site, the evaluation will not appear regardless of this setting.
  • Click Save to commit your changes

Session Presenters

Presenters have administrative control over their session. They can update links, add questions, manage the discussion, etc. Presenters, however, do not have site administration abilities. They cannot edit the schedule, post site announcements or add users (unless they were specifically made site administrators, of course). The list of presenters is shown to users in the Info panel of the session.

Adding a new presenter

  1. Choose the session in the schedule
  2. Click the Admin Tab
  3. In the Presenters section begin typing a name (the user must already exist in the User Directory)
  4. Click add to make the user a presenter. This user will now have the ability to manage this session.

Removing a presenter

  1. Choose the session in the schedule
  2. Click the Admin Tab
  3. In the Presenters section click Remove next to the user you wish to remove

Links

Links can be added to sessions to help share relevant content on the web. By default, any logged in attendee can post links to any session. You can control who can post links by using the Admin Panel in the session.

Adding a link

  1. Choose the session in the schedule
  2. Click the Links tab
  3. Click Add New Link
  4. In the URL box, enter a valid URL, it must contain the protocol (i.e. http://)
  5. Enter the Title, this will inform users what the link is referencing
  6. Click Save

Removing a link

  1. Choose the session in the schedule
  2. Click the Links tab
  3. Click Delete next to the link you wish to remove

Evaluations

If you choose to allow evaluations, users can evaluate each session using the same set of questions.

Viewing evaluation results

  1. Choose the session in the schedule
  2. Click the Admin tab
  3. Click View Evaluations in the options section
  4. Click on each question to view the results for that question. Multiple choice questions will show the number of answers for each response. Free response questions will show each response

Clearing evaluations

  1. Choose the session in the schedule
  2. Click the Admin tab
  3. Click View Evaluations in the options section
  4. Click the Clear Evaluations button. This will clear all evaluations for this session

Questions

Questions can be created to gather immediate response from users. Currently, mobilAP supports multiple choice questions. Presenters can create questions with several possible responses, and can choose how many responses are allowed. The results are shown with a graph and listing of the answers and is constantly updated as users answer the question. Only logged in users can respond to questions and users can only respond once.

Creating a question

  1. Choose the session in the schedule
  2. Click the Questions tab
  3. Click the + button
  4. Enter the question title. Although there is a very high limit to the length of the question, brevity and conciseness are recommended for optimal user experience.
  5. Set the active status of the question. Questions marked no will not appear to users
  6. Select the minimum number of choices the user must answer. If this is set to 0 then the question is optional.
  7. Select the maximum number of choices the user may answer.
  8. Add the possible responses to this question:
    1. Enter a response in the text box
    2. Click Add
    3. Click Remove to remove any responses
  9. Click Save to add the question

Editing/Removing A Question

  1. Choose the session in the schedule
  2. Click the Questions tab
  3. Click the Edit button. This button will not appear if there are no questions
  4. Click Edit next to the question you wish to edit
  5. Update the properties as necessary
  6. Click Save to save your changes
  7. Click Cancel to return to the list without saving
  8. Click Delete to remove this question

Discussion

The discussion area is used for backchannel IM style discussion for the session. Any logged in user can post a short textual statement, and all the posts are ordered chronologically. At this time, the only administrative function is to clear out the discussion

Clearing the discussion

  1. Choose the session in the schedule
  2. Click the Discussion tab
  3. Click the Clear Discussion button. This will clear all posts for this session
