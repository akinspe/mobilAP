mobilAP Documentation - Administration

Once mobilAP has be set up, you can begin adding users and content. In order to administer the site you will need to login using the user you created in the setup routine. All of the steps below assume you have logged in as an administrative user.

mobilAP uses a menu system for each of the sections. You can customize the label of each item as well as removing any items you don't want to see. By default the following menu items are shown:

  • Welcome - An editable page that represents the home page of your installation. You can edit it using standard HTML.
  • Schedule - View and edit the schedule. This also includes adding and editing sessions.
  • Directory - View and edit the directory. Creating and managing users
  • Announcements - View and edit announcements
  • Profile - Shown when a user is logged in. Allows the user to edit portions of their profile and change their password
  • Admin - Configures site wide settings. Only shown when an administrative user is logged in.
  • Login/logout - By default users are kept logged in if they don't specifically log out. This serves as a convenience to users.

It is important to note that only a limited number of administrative functions are available using the mobile interface. All functions are available in the desktop interface.
